PHP 

 

 

Aqui se trabajara la creación del código e inclusión en HTML, verificando que el formulario esté conectado de forma correcta a la BD.

 

PHP (acrónimo recursivo de PHP: Hypertext Preprocessor) es un lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web y que puede ser incrustado en HTML. PHP puede hacer todo tipo de cosas:

 

- Evaluar datos de formularios enviados desde un navegador.

- Construir contenido web a medida para el navegador

- Comunicar a una base de datos.

- Enviar y recibir cookies (pequeños paquetes de datosque tu navegador usa para recordar cosas ejemplo un registro).

 

El intérprete PHP ejecuta los comandos contenidos en el archivo y eventualmente se comunica con un gestor de base de datos (ejemplos de ellos pueden ser Mysql, Oracle, Informix, SQL Server, etc.)

 

Actividad


Para este ejemplo tendremos un formulario que guarda los datos en un registro de usuarios enlazado con una BD ya creada en PhpMyadmin.

El gestor que usaremos para crear la BD es PhpMyadmin, creando una tabla con los siguientes campos: Al crear el formulario ya se debe tener clara la estructura para una página web y la estructura de una tabla aqui una guia. La creación de una tabla la comprende las etiquetas <tr> para las filas y <td> para las columnas, dentro de las columnas estara la etiqueta <input> esta nos permite mostrar un campo de escritura donde el usuario diligenciara la información tener presente darles un nombre que sera guia cuando se guarden los datos. Agregar el codigo PHP sera dentro de las equitetas <?php y ?>. Lo principal es realizar la conexión con la BD y el servidor con las variables $link y $db como se muestra en la imagen. Seguido obtendremos los datos que diligencio el usuario, creando variables que sean precedidas por el caracter $ y el metodo $_POST[' aqui se coloca el nombre que se dio en la tabla '];. La variable $req se coloca para validar que los datos sean todos diligenciados.


Como estamos creando un formulario se debe validar que las contraseñas sean escritas de manera correcta las dos veces, se valida mediante una sentencia if (si) de tal manera que si la primera contraseña es diferente (!=) a la segunda escrita, se muestra una alerta para que se escriba de nuevo. Como son contraseñas deben ser privadas por esto se debe encriptar creando una nueva variable que encriptara con el metodo md5();.

Ahora agregaremos los datos a la BD con el metodo mysql_query dentro de ella el metodo ("INSERT INTO el nombre que tiene la tabla VALUES (' y las variables que se crearon para almacenar los datos ')",$link agregando la conexión y un mensaje si no se pued establecer conexión.


De esta manera si todo es correcto se mostrara con la etiqueta echo un mensaje de registro completado y un agradecimiento.

Habiendo guardado los cambios realizados procedemos a realizar una prueba. El registro a sido exitoso y nos muestra el mensaje que establecimos. Al recargar la BD se evidencia el registro que se ha diligenciado, la clave como evidencia esta encriptada.
